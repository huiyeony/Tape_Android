# 3. Key Features (주요 기능)

- **Create 기능**:
  - 서버에서 불러온 song data 중에서 원하는 곡을 재생목록에 추가합니다.
  - 커버이미지와 제목, 설명을 입력합니다.
  - 생성한 재생목록은 피드에 올라옵니다.
- **Api 인터페이스, 메서드 작성**:
  - 서버와 통신하는 api 인터페이스와 GET/POST/DELETE 메서드를 설계하였습니다.
