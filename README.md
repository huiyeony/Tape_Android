# 3. Key Features (주요 기능)

- **Create 기능**:
  - RoomDb Database 에 저장된 Dummby Data 중에서 원하는 곡을 재생목록에 추가합니다.
  - Tape의 커버이미지와 제목, 내용을 입력합니다.
  - 생성한 재생목록은 홈 피드에 올라옵니다.
- **Api 인터페이스, 메서드 작성**:
  - REST Api GET/POST/DELETE 메서드로 서버와 통신할 수 있습니다.
